# SnapScrap.py
Downloads public Snapchat stories in your System.

## Saving this locally
```bash
$ git clone https://github.com/allendema/SnapScrap.py.git
$ pip3 install -r requirements.txt
```

## Run it
```bash
$ ./SnapScrap.py
```
## Use it
Just enter one username which has Public Profile.
